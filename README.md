## `01` · Engineering Identity

Hardware Design Engineer with production experience in **power electronics, EVSE systems, and multi-layer PCB design** — actively deepening expertise in **Signal Integrity (SI), Power Integrity (PI), and High-Speed Digital Design**.

My engineering approach is grounded in understanding *why* failures happen at the physics level — return current paths, impedance discontinuities, simultaneous switching noise, and PDN collapse — not just correcting symptoms.

**Current:** Winet Infratel R&D — 3.3kW & 7.4kW EVSE Charger Systems  
**Academic:** B.Tech — Electronics & Communication Engineering  
**Direction:** Signal Integrity · Power Integrity · High-Speed PCB Architecture

## `05` · Flagship Projects

<details>
<summary><b>⚡ 7.4 kW EVSE Charger System</b> — Industry | Winet Infratel</summary>
<br>

**System Overview:** Type-2 AC EV charger conforming to IEC 61851 / IEC 62196 standards

| Parameter | Detail |
|---|---|
| Output Power | 7.4 kW (single-phase) |
| Control Interface | Control Pilot — PWM duty-cycle per SAE J1772 |
| Controller | ESP32-based control PCB |
| Protection | GFCI, OVP, OCP, relay feedback monitoring |
| PCB Scope | Multi-board system: control + power stages |

**Engineering Decisions:**
- Isolated control pilot circuit with precision duty-cycle generation
- MOSFET + relay switching with feedback-verified state machine
- PCB layout optimized for clearance, creepage, and thermal dissipation
- Hardware debugging via oscilloscope validation of pilot waveforms

</details>

---

<details>
<summary><b>⚡ 120W Synchronous Buck Converter</b> — LM5116 | High-Efficiency DC-DC</summary>
<br>

| Parameter | Detail |
|---|---|
| Controller IC | Texas Instruments LM5116 |
| Output Power | 120W |
| Topology | Synchronous Buck |
| Key Focus | Feedback loop compensation, PCB noise minimization |

**Engineering Decisions:**
- Type-III compensation network designed for phase margin optimization
- Kelvin sensing at output for accurate regulation
- Minimized high-frequency loop area to reduce switching noise coupling
- Verified efficiency curve via electronic load testing

</details>

---

<details>
<summary><b>⚡ 60W Flyback SMPS</b> — Isolated Power Supply</summary>
<br>

| Parameter | Detail |
|---|---|
| Output Power | 60W |
| Topology | Flyback (isolated) |
| Feedback | Optocoupler + TL431-based error amplifier |

**Engineering Decisions:**
- Transformer design with defined turns ratio for target output
- Snubber network calculated for leakage spike suppression
- PCB primary/secondary separation for isolation integrity
- Verified regulation under load step with oscilloscope capture

</details>

---

<details>
<summary><b>🔋 2S Battery Management System</b></summary>
<br>

| Parameter | Detail |
|---|---|

| Protections | OVP, UVP, OCP |
| Switching | MOSFET-based charge/discharge paths |

**Engineering Decisions:**
- Dual N-MOSFET back-to-back for bidirectional protection
- Dedicated sense resistor for overcurrent detection
- Layout reviewed for low-resistance current paths and thermal headroom

</details>
